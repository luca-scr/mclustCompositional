# mclustCompositional

An [R](https://www.r-project.org/) package  implementing model-based clustering and density estimation for compositional data using transformation-based Gaussian mixtures.

## Installation

- Install/update `pak` package to the latest version

  ```
  install.packages("pak")
  ```

- Install/update `mclustCompositional` package from GitHub

  ```
  pak::pkg_install("luca-scr/statml")
  ```
